Download using Git

You can install the theme by cloning the repo:

$ git clone https://github.com/mateusortiz/sunnyvale-theme/

Download manually

Download GitHub .zipand unzip them.

Activating the Theme

Just locate the User Stylesheets directory and override the Custom.css:

  1. Mac OS X: Replaces chrome/Custom.css file to ~/Library/Application\ Support/Google/Chrome/Default/User\ StyleSheets/Custom.css

  2. PC(Windows Vista, 7, 8) : Replaces chrome/Custom.css file to C:\Users\YourUsername\AppData\Local\Google\Chrome\User Data\Default\User StyleSheets\Custom.css

  3. Ubuntu (Chromium) : Replaces chrome/Custom.css file to ~/.config/chromium/Default/User\ StyleSheets/Custom.css
